== Description ==

Easily gather feedback, conduct market research, or engage your audience with the **Simple Poll & Survey Builder**. This lightweight and user-friendly plugin allows you to create customized polls in minutes without any coding knowledge.

Whether you want to know your customers' preferences or run a quick vote on a hot topic, this plugin provides a seamless experience for both site admins and visitors.

**Key Features:**

* **Easy Setup:** Create new polls through an intuitive interface.
* **Shortcode Support:** Display your polls anywhere—posts, pages, or widgets.
* **Lightweight:** Built with performance in mind to ensure fast loading times.
* **Mobile Responsive:** Polls look great on desktops, tablets, and smartphones.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the 'Polls' menu to create your first survey.
4. Copy the shortcode and paste it into any page or post.

== Frequently Asked Questions ==

= How do I display a specific poll? =
Every poll has a unique ID. Simply use the shortcode `[my_poll id='1']`. Replace '1' with your actual poll ID.

= Can I use multiple polls on one page? =
Yes, you can add multiple shortcodes to a single page or post as needed.
